## What was built

### Product surface
- Static single-page web app with a branded landing page and embedded working tracker
- Local-first storage via `localStorage`
- Session logging for product, strain, dose, method, time, context, notes, and five effect sliders
- Insight engine for sleep, pain, calm, balance, and average effect profile
- Tolerance-break mode with streak progress, symptom logs, and re-entry protocol
- JSON export, encrypted backup export/import, and doctor CSV export

### Packaging work
- Build-only CI workflow at `.github/workflows/ios-build.yml`

- Service worker at `sw.js`
- Capacitor config at `capacitor.config.json`
- Native iOS shell added under `ios/`
- iOS display name confirms as `Moss Journal` in `ios/App/App/Info.plist`

- Check all four tabs: Journal, Insights, T-break, Backup
